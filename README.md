## 使用方法

- 将文件夹"patch2"打包成xp3文件后，放入游戏根目录即可；__或者您可以直接使用[发行版本](https://github.com/noodle-MKII/Dungeon_and_Bride_CHS_Translate/releases)__
- 推荐使用"game64.exe"进入游戏
